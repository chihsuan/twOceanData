# twOceanData
Crawl the taiwan ocean data (海洋水質、測站)

## Source

* [海洋水質監測與指標](http://edw.epa.gov.tw/reportInspectSea.aspx)
* 海洋測站資訊網頁: `http://edw.epa.gov.tw/ResultSeaSite.aspx?siteid={siteid}`

## require
+ Install nodejs
+ Install npm

## Usage

first, type
    
    $ npm install
    $ node oceanData.js
    $ output: data.json

then,

    $ node siteData.js
    $ output: site.json
